# Security Policy

## Supported Versions

| Version | Supported          |
| ------- | ------------------ |
| latest  | :white_check_mark: |

## Reporting a Vulnerability

If you discover a security vulnerability in this project, please report it responsibly:

1. **Do not** open a public GitHub issue for security vulnerabilities.
2. Email the maintainer or use [GitHub's private vulnerability reporting](https://github.com/cheelim1/ecs-autoscaler/security/advisories/new).
3. Include a description of the vulnerability, steps to reproduce, and any potential impact.
4. You can expect an initial response within 72 hours.

We appreciate your help in keeping this project secure.
